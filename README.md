
#Algo-lab

This repository contains my work for the Algorithm and Problem Solving Lab course.
All programs and experiments are implemented using Python in Google Colab.

About

The aim of this lab is to understand basic algorithms and problem-solving techniques.
Each notebook contains solutions to lab problems along with required outputs.

Repository Contents

Lab_*.ipynb – Lab programs and solutions

README.md – Description of the repository

How to Run

Open https://colab.research.google.com

Click File → Open notebook

Go to the GitHub tab

Paste this link:

https://github.com/AryanAnand-ux/Algo-lab.git


Open any notebook and run the cells

No installation is required.
